# askip-stmg-calculs-sdgn
Askip’en STMG Calculs est une plateforme pédagogique interactive destinée aux élèves de STMG, conçue pour les entraîner aux calculs fondamentaux et aux raisonnements méthodologiques exigés au lycée et au baccalauréat.  L’outil vise un objectif simple : aider les élèves à savoir quelles données sélectionner, quoi calculer et comment raisonner.
